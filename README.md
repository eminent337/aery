# Aery

> AI coding agent for the terminal. Built by Aryee.

## Install

```bash
npm install -g @eminent337/aery
```

## Packages

| Package | Description |
|---------|-------------|
| `packages/coding-agent` | Aery CLI — the main agent |
| `packages/agent` | Agent core — loop, tools, session |
| `packages/ai` | AI providers — 300+ models |
| `packages/tui` | Terminal UI |
| `packages/mom` | Slack integration |
| `packages/pods` | GPU deployment manager |
| `packages/web-ui` | Web UI components |

## Development

git clone https://github.com/eminent337/aery && cd aery && ./install.sh
