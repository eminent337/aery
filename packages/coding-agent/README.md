# Aery

> AI coding agent for the terminal. Built by Aryee.

## Install

```bash
npm install -g @eminent337/aery
```

Or from source:

```bash
git clone https://github.com/eminent337/aery && cd aery && ./install.sh
```

## Quick Start

```bash
aery
```

## Extensions

All features are added as extensions in `~/.aery/agent/extensions/`.

## Configuration

Settings: `~/.aery/agent/settings.json`
